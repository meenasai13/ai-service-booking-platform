# AI Service Booking Platform

AI-based web application for booking service providers like electricians, plumbers, and cleaners. The platform allows users to register, log in, view providers, book services, manage bookings, and get AI-based provider recommendations using Machine Learning.

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Python Flask
- MySQL
- Machine Learning (KNN)
- REST APIs
- Scikit-learn

---

## Features

- User Registration
- User Login
- View Service Providers
- Book Providers
- Booking History
- Cancel Booking
- AI Recommendation System
- AI Assistant Chatbot
- Dashboard
